Instructions

Dans cet index.html met en page un résultat de recherche de voyage avec les données disponibles dans le rendu React dans cette page là, dans un délai d'une heure.

Si tu as besoin des données brutes, tu peux les trouver dans le data.json

Au préalable un wireframing ou une maquette sur Xd (ou un autre outils dédié au web) serait la bienvenue, fournit en retour de ce test.

Reste concentré sur les données, il n'y a que ça à traiter (pas de header, footer, nav...).

Tu disposes de normalize css, fontawesome free, jQuery css (déjà installés mais pas obligatoire).

Pour le style, tu as la possibilité d'utiliser soit Less (dossier less) ou Sass (dossier scss).

Une fois terminé (au bout d'une heure) push ta branche prenom.premiere_lettre_nom
